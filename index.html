Xemulartor no es un emu lador en si. sino una aolicación aue sirve de base para instalar en ella diferentes emu ladores v ejecutar con ellos las ROMs. Por lo tanto, para poder utilizar los juegos de determinada consola no vale con sólo descargar emulator, sino que despues tendras que cargar en la aplicacion el emulador correspondiente. Esta aplicación es la solución de referencia basada en LibRETRO, una sencilla API que permite la creacion de juegos y emuladores. Se trata de un sistema modular que permite cargar diterentes nucleos, cada uno de ellos con un emulador, todc ello pudiendose nacer desde la propia aplicacion. Ademas de RetroArch, en esta tecnologia tambien se basan otros proyectos.
